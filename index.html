Proyectos

Sistema de Información para la UMADIS de Punata

Desarrollé un sistema de información para la UMADIS (Unidad Municipal de Atención a Personas con Discapacidad) del municipio de Punata, diseñado para almacenar, gestionar y analizar los datos de personas registradas. El sistema permite la captura eficiente de información personal, administración de los registros y la aplicación de filtros avanzados basados en diferentes características, como el tipo de discapacidad, grado de discapacidad, entre otros. Además, incluye un módulo para generar estadísticas y gráficos sobre los datos almacenados.

Página Web Laboratorio de Geotecnia

Página web para el laboratorio de Geotecnia de la Universidad Mayor de San Simón utilizando WordPress, diseñada para ofrecer información detallada sobre los servicios y recursos del laboratorio.

Gestion de Tienda Online

Plataforma Web de comercio electrónico completa que permite a los usuarios vender productos, administrar inventarios, y optimizar las ventas con diversas funcionalidades avanzadas. La tienda cuenta con un sistema de gestión de productos que permite añadir, editar y eliminar artículos, así como gestionar cantidades en tiempo real. También incluye la opción de aplicar descuentos y rebajas automáticas, mejorando la experiencia del cliente.

Plataforma de Alojamiento Tipo Airbnb

Una réplica funcional de Airbnb, diseñada para facilitar la búsqueda, reserva y administración de alojamientos temporales. La plataforma permite a los usuarios registrarse como anfitriones para publicar propiedades o como huéspedes para buscar y reservar alojamiento. La interfaz es intuitiva, con filtros avanzados para la búsqueda por ubicación, precio, tipo de propiedad y fechas.
